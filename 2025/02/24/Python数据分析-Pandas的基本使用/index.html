Pandas的数据类型

Series

Series是一个一维的类似数组的对象，包含一个数组的数据（任何NumPy的数据类型）和一个与数组关联的数据标签，被叫做索引（index） 。

默认的情况下索引是0到N-1的整数。

Series.values返回的是该Series的ndarray表示；Series.index返回的是该Series的索引对象。

自定义索引可以采取下面的办法：

比起NumPy的数组类型，Series可以根据索引的值来选取相应的值或值集合；

对Series也可以进行与NumPy数组类似的操作，布尔数组过滤，纯量乘法以及使用数学函数，不同的是索引和值的联系将会保持。

Series也可以看作是一个定长的有序Python字典类型，可以运用在字典上的函数也可以同样地作用在Series上。

Series还具备有isnull()和notnull()方法，返回布尔Series。

Series在数据分析中的一个显著的功能就是不同的Series在进行运算的时候，会按照索引实现自动对齐。

Series和他的索引都具备有name属性。

Series的索引可以通过赋值就地进行更改

DataFrame

DataFrame是一个二维表格，具有行索引和列索引。

构造DataFrame

可以传给DataFrame构造器的对象很多，包括：
二维ndarray，数组、列表或元组的字典，Series的字典，字典的字典，字典或者Series列表，列表或元组的列表，另一个DataFrame，NumPy的伪装数组。

DataFrame的索引

注意pandas的索引对象是不可变的，可以包含有重复值。

DataFrame中的一列可以通过字典记法来检索（即使用中括号）。

DataFrame中的一行使用DataFrame.loc属性来索引成员。

==DataFrame的loc属性很有用，后面还要用到。==

DataFrame的赋值

DataFrame的列可以通过赋值来修改，可以由纯量或者一个数组来赋值。给不存在的列赋值会创建一个新列。数组赋值需要保证长度匹配。此外，还可以通过Series来赋值，这时会按照索引对齐，空缺位置会被填充NaN。

del关键字会删除列，就像字典一样。

如同NumPy，索引DataFrame返回的都是原数组的一个视图，而非拷贝，显式的拷贝需要使用copy方法。

DataFrame的列和行的columns和index也都有它们的name属性。

DataFrame在进行算数的时候也好进行数据的自动对齐。

Pandas的基本功能

重新索引

在Series上调用 reindex 重排数据，使得它符合新的索引，如果那个索引的值不存在就引入缺失数据值。

有时候可以指定reindex()函数的的method参数为‘ffill’或者是‘bfill’分别表是向前填充和向后填充。

只给reindex()函数传进去一个列表的话，默认是给行做重新索引，要给列做重新索引的话，可以指定‘columns=[….]’

用loc方式也可以进行重新索引：
frame.loc[[‘a’, ‘b’, ‘c’, ‘d’], [‘pop’, ‘year’]]
这很方便，也非常有用。

删除

Series和DataFrame都有drop方法。指定Series的索引或者索引列表，可以删除相应的元素；对于DataFrame直接传入的索引或索引列表默认是对行进行操作，要指定对列进行操作的话，传入参数axis=1或axis=‘columns’。同时传入参数inplace=True将会在原数据框中修改数据，而不是返回一个新的对象。

inplace = True 一定要谨慎，数据删掉了就回不来了。如果不能保证代码一次性就能写对的话，不要加上，对于调试过程非常不友好。

索引、选择和过滤

Series的索引和NumPy差不多，可以使用整数来索引和切片。不同的是也可以使用标签来切片，不同的是此时的切片是左闭右闭的（很容易想到这是为什么）。选取后进行赋值的操作和NumPy也是类似的。

DataFrame的索引默认是对列做的。

这里当然还能够使用loc方法和iloc方法来进行选择，不再赘述。

函数应用与映像

NumPy中的ufunc可以直接应用到pandas对象上。例如frame是一个pandas的DataFrame，就可以对其应用np.abs(frame)，将每个元素取绝对值。

如若是要对DataFrame的每个维度应用一个函数，需要使用apply方法。默认情况下是对列做的，如若要对行做，指定参数axis=1或者axis=‘columns’。

用applymap方法可以将函数作用到DataFrame的每个元素上。
用map方法可以将函数作用到Series的每个元素上。

排序与排名

pandas内置了排序功能，sort_index可以按照索引排序，sort_values可以按照值进行排序。

rank方法会得到Series的一个排名Series，对于大小相同的值，该取值会取排名的平均。

描述性统计

求和：df.sum()
累积求和：cumsum()
最大最小值的索引：df.idxmax(), df.idxmin()
非规约非累积方法：df.describe()
相关系数：Series.corr()，返回相关系数矩阵
协方差：Series.cov()，返回协方差矩阵
唯一值：Series.unique()

等等，可以现用现查。
